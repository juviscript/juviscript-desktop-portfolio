# Juviscript.dev Portfolio - Project Requirements

## Project Overview
Retro Windows 98-style (but cuter) interactive portfolio showcasing development work, 
skills, and personality through an 8-bit desktop environment living in the browser.

## Goals
- Make it FUN for me and the user.
- Leave a lasting impression.
- Showcase technical skills.
- Mobile-responsive!
- Easy to update with new projects.

## Target Audience
- Hiring managers / recruiters.
- Other Developers.
- Potential clients.

## Must-Have Features (MVP)
- [ ] Boot sequence animation (BIOS then logo load) that is skippable
- [ ] Desktop with icons
- [ ] Draggable windows
- [ ] Projects explorer (3-4 projects minimum)
- [ ] About me section
- [ ] Contact form (functional)
- [ ] Resume download
- [ ] Sound effects
- [ ] Certifications display
- [ ] Mobile-responsive version

## Nice-to-Have Features (Post-MVP)
- [ ] Tech stack visualization
- [ ] Easter eggs (Recycle Bin with silly items deleted, pictures of my corgi, right-click menu, mini-games!)
- [ ] Windows Media Player visualizer
- [ ] Links to Socials that open within the portfolio, and then can be opened in the browser. 
- [ ] Start menu
- [ ] Themes and customizable backgrounds that are saved in local storage.

## Technical Constraints
- ACCESSIBLE (keyboard navigation where possible)
- Must load quickly (<3 seconds)
- Works on Chrome, Firefox, Safari
- SEO-friendly

## Out of Scope (For v1)
- User accounts / authentication
- Blog functionality
- Admin panel
- Multi-language support
